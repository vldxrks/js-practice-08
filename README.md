## Мета
Опанувати використання функцій вищого порядку, композицію функцій, чисті функції та функціональне програмування в JavaScript.

## Опис реалізованих функцій

У межах цього проєкту створено бібліотеку функцій вищого порядку для трансформації даних. Усі реалізовані функції є чистими (не мають побічних ефектів) та підтримують можливість створення ланцюжків викликів (chainable API).

### 1. Базові трансформації даних
* **`map`** — власна реалізація методу для створення нового масиву з результатами виклику вказаної функції для кожного елемента.
* **`filter`** — власна реалізація методу для створення масиву з усіма елементами, які проходять перевірку заданою функцією.
* **`reduce`** — власна реалізація методу для застосування функції до акумулятора та кожного значення масиву (зліва направо), зводячи його до одного значення.

### 2. Композиція
* **`compose`** — функція для композиції інших функцій (виконується справа наліво).
* **`pipe`** — функція для композиції інших функцій (виконується зліва направо).

### 3. Функціональні утиліти
* **`curry`** — функція для каррінгу довільної функції (дозволяє викликати функцію з меншою кількістю аргументів, повертаючи нову функцію, яка очікує наступні аргументи).
* **`partial`** — часткове застосування (дозволяє зафіксувати частину аргументів функції наперед).
